## Learning Vim Script Now
----------------
* Boolean Options 
> set <name> turns the option on and :set no<name> turns it off.     
> ```:set nonumber``` 
----------
    
* Toggling Boolean Options  
> Adding a ! (exclamation point or "bang") to a boolean option toggles it.    
> ```:set number!``` 
----------
    
* Checking Options 
> You can ask Vim what an option is currently set to by using a ?       
> ```:set number?```
